# Roadmap Short Form

## Phase 0 — Design Lock and Project Setup

Current phase. Establish project structure, iPhone-first web target, Phaser/Vite foundation, placeholder-safe content, and deployment path.

## Phase 1 — Menu, Save, and Hub Skeleton

Build real menu flow, options, customization placeholder, gallery placeholder, and progress-driven hub state.

## Phase 2 — Side-Scroller Core

Build movement, jump, attack, dodge, basic enemies, pickups, level bounds, and clothing-health.

## Phase 3 — Engagement System Prototype

Build trap/elite engagement states, struggle meter, tap escape, partial/full failure outcomes, and unlock flags.

## Phase 4 — Boss Combat Prototype

Build boss HP, boss energy, phases, skills, boss engagement special, and win/loss states.

## Phase 5 — Ending Resolver and Unlock System

Connect outcomes to route flags, gallery unlocks, skills, and hub updates.

## Phase 6 — Scene Viewer and Gallery v0.1

Build gallery categories and phase-based scene playback controls.

## Phase 7 — Customization and Progression v0.1

Connect outfits, passives, skills, and customization unlocks.

## Phase 8 — First Public Playable Build

Publish iPhone-friendly GitHub Pages prototype.

## Phase 9 — Content Expansion

Add more levels, enemies, traps, elites, bosses, endings, and unlocks.

## Phase 10 — Polish and Balancing

Improve UI, effects, animation, performance, accessibility, balancing, and save tools.
